# 야외 연습 결과 점검

세션이 끝났다면 이제 결과를 열어보고 관찰·진단·개선하는 복기 루프를 돌 차례입니다. 이 페이지는 "할 수 있다 / 안다"를 확인하는 사전 능력 체크가 아니라, 세션이 끝난 뒤 "무엇이 잘못됐고 다음엔 어떻게 고칠까"를 찾는 과정입니다.

## 증상별 점검

아래 표를 따라 이번 세션 결과물을 하나씩 열어보며 확인하세요.

| 증상 | 확인·원인 | 다음 개선 |
|---|---|---|
| 별이 궤적으로 번졌다 | LCD·모니터에서 100% 확대로 다시 확인. [500 / NPF 룰](../02-fundamentals/500-npf-rule.md)로 셔터속도를 다시 계산했는지 점검 | 계산한 상한보다 더 짧은 셔터속도로 재시도 |
| 초점이 흐리다 | [별에 초점 맞추기](../02-fundamentals/focusing.md)의 방법대로 100% 확대해 별이 점으로 남았는지 확인 | 다음 세션에서 라이브뷰 확대·미세조정 단계를 더 천천히, 테이프 고정까지 반복 |
| 노출이 어긋났다(너무 어둡거나 하이라이트가 날아갔다) | [현장 촬영 워크플로](../02-fundamentals/field-workflow.md)의 시험 촬영·히스토그램 확인 단계를 실제로 거쳤는지 점검 | 다음 세션에서는 본 촬영 전 히스토그램 확인을 빠뜨리지 않기 |
| 사진이 뿌옇다 | 이슬·김서림 여부 확인 — [야외 준비와 현장 워크플로 리허설](field-prep-and-hazards.md) | 다음 세션에서 세션 중 렌즈 앞을 주기적으로 닦는 습관 강화 |
| 수평이 기울었다 | 삼각대 수평계 재확인 | 다음 세션에서 구도 잡기 전 수평계부터 켜기 |
| 구도가 아쉽다 | 전경 배치, 코어 방향과의 정렬을 다시 검토 | 다음 세션에서 밝을 때 미리 전경 위치를 스카우팅 |

## template.md로 이어가기

위에서 관찰한 내용을 [연습 로그 템플릿](template.md)의 **"배운 점 / 문제점"** 섹션에 적고, 그중 다음에 고칠 것들을 **"다음 세션 개선 사항"** 섹션에 정리하세요. 이 항목이 그대로 다음 세션 로그의 "목표"로 이어집니다 — 매번 새로 시작하는 것이 아니라, 이렇게 세션마다 이전 문제를 하나씩 지워나가는 방식으로 실력이 쌓입니다.
