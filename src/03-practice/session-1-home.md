# 연습 1차 — 집 (2026-MM-DD)

이번 세션은 은하수를 찍는 연습이 아니라, [현장 촬영 워크플로](../02-fundamentals/field-workflow.md)의 순서를 집에서 미리 예행연습하며 세팅·초점·워크플로를 몸에 익히는 홈 리허설이다.

## 목표

- M모드로 전환해 노출 3요소(ISO / 조리개 / 셔터스피드)를 직접 조작해본다 → [노출의 기본](../02-fundamentals/exposure-basics.md) 참조.
- 본인 카메라·렌즈의 크롭계수를 [500 / NPF 룰](../02-fundamentals/500-npf-rule.md)에서 확인해 최대 셔터속도를 **직접 계산**해 목표값으로 적어둔다 (마운트·렌즈가 아직 확정되지 않았으므로 숫자를 미리 적지 않고, 본인 장비 기준으로 그때그때 계산할 것).
- 별이 아닌 **먼 가로등·간판 불빛**을 대상으로 라이브뷰 10배 확대 → 수동 초점 → 테이프 고정까지 전체 절차를 미리 연습한다 → [별에 초점 맞추기](../02-fundamentals/focusing.md)의 방법을 그대로 따른다(밤하늘 아래서 처음 시도하지 말라는 원칙이 이 페이지에 명시되어 있다).
- RAW 저장 + 수동 화이트밸런스 고정(3800~4200K) 설정을 확인한다 → [RAW와 화이트밸런스](../02-fundamentals/raw-and-wb.md).
- 인터벌 타이머/리모컨으로 셔터를 손으로 누르지 않고 촬영하는 법을 확인한다.
- (선택) 삼각대 설치·수평, 시험 촬영 후 100% 확대해 불빛이 점으로 맺히는지 확인하는 습관.
