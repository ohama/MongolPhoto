# 카메라 조작·세팅 연습

현장은 어둡고 춥고 시간이 넉넉하지 않습니다. [노출의 기본](../02-fundamentals/exposure-basics.md), [500 / NPF 룰](../02-fundamentals/500-npf-rule.md), [RAW와 화이트밸런스](../02-fundamentals/raw-and-wb.md)에서 배운 이론을 실전에서 헤매지 않고 바로 쓰려면, 집에서 미리 손에 익혀 두는 것이 목표입니다.

## 연습 항목

1. **M모드 다이얼 감각 익히기 (어둠 속 조작)** — M모드로 전환한 뒤, 화면을 보지 않거나 실내조명을 끈 상태에서 다이얼만으로 ISO·조리개·셔터스피드를 바꿔 봅니다. 실제 밤하늘 아래에서는 화면이 거의 안 보이고 손끝 감각으로 조작해야 하는 경우가 많습니다.
2. **본인 장비로 크롭 셔터 상한 직접 계산** — [500 / NPF 룰](../02-fundamentals/500-npf-rule.md)의 공식에 본인 카메라·렌즈의 초점거리와 크롭계수를 대입해 최대 노출시간을 직접 계산하고, 그 값을 메모해 둡니다. 마운트·렌즈마다 값이 다르므로 반드시 본인 장비 기준으로 계산하세요.
3. **RAW + 수동 화이트밸런스 설정 확인** — [RAW와 화이트밸런스](../02-fundamentals/raw-and-wb.md)의 안내대로 파일 형식을 RAW로, 화이트밸런스를 3800~4200K 수동으로 맞춰 봅니다.
4. **전원·저장 여유 확인** — [출발 전 체크리스트](../01-gear/checklist.md)의 배터리·메모리 카드 항목을 미리 훑어 봅니다.

## 연습 방법

한 번에 다 익히려 하지 말고 며칠에 걸쳐 반복하세요. 목표는 생각하지 않고도 손이 먼저 움직이는 수준입니다. 실전은 어둡고 춥고 시간이 없어서, 여기서 몸에 익힌 만큼 현장에서 여유가 생깁니다.
