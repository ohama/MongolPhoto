# 집 연습 체크리스트

[카메라 조작·세팅 연습](home-camera-controls.md)과 [초점·구도·인터벌 연습](home-focus-and-interval.md)을 마쳤다면, 아래 항목으로 스스로 점검해 보세요. 모두 결과 중심으로 적었으니 "할 수 있다 / 안다" 기준으로 체크하면 됩니다.

## 조작·세팅

- [ ] 화면을 보지 않고도 M모드에서 ISO·조리개·셔터를 다이얼로 바꿀 수 있다
- [ ] 내 카메라·렌즈 기준 최대 셔터속도를 계산해 적어 두었다
- [ ] RAW + 수동 화이트밸런스(3800~4200K)로 설정할 수 있다
- [ ] 배터리·메모리 카드가 얼마나 필요한지 감이 있다

## 초점·구도

- [ ] 먼 불빛에 라이브뷰 확대 수동 초점을 맞출 수 있다
- [ ] 테스트 샷을 100% 확대해 초점이 맞았는지 스스로 확인할 수 있다
- [ ] 삼각대 수평을 맞추고 세로·가로 구도를 잡을 수 있다

## 촬영 진행

- [ ] 인터벌 타이머/릴리즈로 셔터를 직접 누르지 않고 여러 장 찍을 수 있다
- [ ] 촬영 직후 히스토그램을 확인하는 습관이 있다

모두 체크되면 [야외 연습](field-practice.md)으로 넘어갈 준비가 된 것입니다.
