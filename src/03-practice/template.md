# 연습 로그 템플릿

아래는 매 연습 세션마다 그대로 복사해서 쓰는 기록 양식입니다. [2부 · 가기 전 연습](home-camera-controls.md)의 안내를 따라 연습하면서 이 양식에 기록하고, 함께 연습하는 지인도 각자 자기 로그에 이 템플릿을 그대로 복사해 쓸 수 있습니다. 사진이나 촬영 환경이 없는 항목은 비워두고 넘어가도 됩니다 — 중요한 건 매번 같은 틀로 기록해서 다음 세션과 비교할 수 있게 만드는 것입니다.

## 사용법

1. 이 페이지의 마크다운 소스를 복사합니다.
2. `연습 N차 — [장소] ([날짜])` 제목을 채웁니다.
3. 세션이 끝난 뒤(또는 끝나고 며칠 안에) 6개 섹션을 채웁니다.
4. 마지막 "다음 세션 개선 사항"에 적은 내용이 **다음 세션의 "목표" 섹션**으로 그대로 이어집니다 — 이렇게 세션마다 이전 실수를 순차적으로 고쳐 나갑니다.

## 템플릿

```markdown
# 연습 N차 — [장소] ([날짜])

## 목표
이번 세션에서 확인/연습하려는 것 (예: 초점 잡는 법 숙달, 인터벌 타이머 세팅)

## 장비 & 세팅
사용 장비, 카메라 설정값 (조리개 / ISO / 셔터스피드 / 초점거리 / 화이트밸런스)

## 촬영 환경
날씨, 월령(달 위상), 광공해(Bortle), 시간대 (야외 세션이면 위치·GPS 추가)

## 결과 사진
![설명](../images/practice/session-N/xxx.jpg)
*그림. 캡션*

## 배운 점 / 문제점
잘 된 것, 실패한 것, 원인 분석

## 다음 세션 개선 사항
다음 번에 바꿀 것 → 다음 세션 "목표" 섹션과 연결됨
```

## 섹션별 채우는 요령

- **목표**: 한두 문장으로 충분합니다. "이번엔 초점만 확실히 맞춰본다"처럼 구체적일수록 좋습니다.
- **장비 & 세팅**: [현장 촬영 워크플로](../02-fundamentals/field-workflow.md) 4단계에서 정한 값을 그대로 옮겨 적으면 됩니다.
- **촬영 환경**: [은하수 찾기와 타이밍](../02-fundamentals/finding-the-milkyway.md)의 체크리스트(계절/시간/방향/달/Bortle/날씨)를 그대로 활용해 기록합니다.
- **결과 사진**: 이미지 경로는 `../images/practice/session-N/` 형태의 상대경로 예시를 참고합니다. 실제 세션이 끝나기 전에는 이미지 파일을 만들지 않습니다 — 사진은 촬영이 끝난 뒤 실제 결과물로 채웁니다.
- **배운 점 / 문제점**: 잘 된 것과 실패한 것을 모두 적습니다. 실패 원인까지 한 줄로 적어두면 다음 섹션(다음 세션 개선 사항)이 훨씬 쉬워집니다.
- **다음 세션 개선 사항**: 이 항목이 다음 세션 로그의 "목표"가 됩니다. 이 연결 고리가 이 템플릿의 핵심입니다 — 매번 새로 시작하는 게 아니라, 이전 세션의 문제를 하나씩 지워나가는 방식으로 연습이 쌓입니다.
