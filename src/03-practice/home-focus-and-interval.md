# 초점·구도·인터벌 연습

## 1. 초점 연습 — 먼 불빛에 무한대 맞추기

[별에 초점 맞추기](../02-fundamentals/focusing.md)의 방법을 그대로 적용하되, 별 대신 가로등·건너편 아파트나 간판 불빛·먼 전파탑 조명처럼 **가능한 한 멀리 있는 점광원**을 대상으로 연습합니다. 단계별 방법은 focusing.md에 이미 있으니 다시 옮기지 않고 그대로 따라 하세요.

**성공 기준:** 라이브뷰 10배 확대 화면에서 불빛이 작고 또렷한 점으로 보이고, 테스트 샷을 100% 확대해도 여전히 점으로 남으면 됩니다.

focusing.md도 이 절차를 "반드시 실제 밤하늘 아래서 처음 시도하지 말라"고 권합니다 — 집 발코니나 창밖에서 먼저 감을 잡아 두세요.

## 2. 구도 연습 — 프레이밍·수평 잡기

창밖이나 베란다에서 "전경 + 하늘(또는 먼 불빛)" 구도를 세로·가로로 각각 잡아 봅니다. 전자 수평계(카메라 라이브뷰 화면이나 메뉴에서 켤 수 있는 수평 표시 기능)를 켜고 삼각대 수평을 맞추는 연습도 함께 해 두세요. 현장에서는 어두워서 수평이 틀어진 것을 눈으로 알아채기 어렵습니다.

## 3. 인터벌 타이머 / 릴리즈 연습

[필수 액세서리 · 앱](../01-gear/accessories.md)에서 소개한 인터벌 타이머(바디 내장 또는 릴리즈)를 실제로 설정해, 셔터를 손으로 누르지 않고 여러 장을 연속으로 촬영하는 연습(드라이런)을 해 봅니다. 셔터 진동을 피해야 하는 이유는 accessories.md에 이미 설명되어 있으니 여기서는 다시 다루지 않습니다. 촬영 직후 히스토그램을 확인하는 습관도 이때 같이 들여 두세요.
