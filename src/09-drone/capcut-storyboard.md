# 예시 편집 — 고비 드론 스토리보드

앞의 장들에서 익힌 편집 단계를 하나로 이어, 대표적인 고비 드론 클립으로 한 편의 짧은 영상을 완성해 보는 예시입니다. 이 장은 가져오기부터 내보내기까지를 스텝바이스텝으로 따라가도록 안내할 예정입니다.

## 이 장에서 다룰 내용

- 클립 가져오기 — 대표 고비 클립을 프로젝트로 불러오기
- 컷 — 쓸 장면만 골라 순서대로 배치하기
- 색보정 — 사막의 색과 빛을 자연스럽게 다듬기
- 음악 — 분위기에 맞는 배경음악 얹기(라이선스 확인 포함)
- 내보내기 — 공유용 파일로 마무리하기

> 📝 이 장은 **작성 예정**입니다. 대표 시나리오와 저자의 실제 촬영본·편집 화면 캡처는 트립(8/13) 후 저자가 직접 촬영·캡처해 채웁니다.
