# 내보내기 설정

편집을 마쳤다면 하나의 영상 파일로 내보내야 공유할 수 있습니다. 이 장은 내보내기 옵션이 결과물에 어떤 영향을 주는지와, 유튜브 업로드를 염두에 둔 설정 방향을 안내할 예정입니다.

## 이 장에서 다룰 내용

- 해상도 선택 — 원본 화질을 살리는 기준
- 프레임레이트 — 촬영 프레임레이트와 맞추는 원리
- 비트레이트 — 화질과 파일 크기의 균형
- 유튜브 업로드용 권장 설정의 방향

> 📝 이 장은 **작성 예정**입니다. 구체 권장 수치는 확인 후 기입할 예정이며, 현재는 다룰 주제만 잡아 두었습니다. 실제 화면 캡처는 트립 후 저자가 직접 캡처합니다.
