# 색보정과 음악

같은 영상도 색과 음악에 따라 분위기가 크게 달라집니다. 이 장은 CapCut에서 색을 다듬는 기본과, 음악을 얹을 때 반드시 짚어야 할 라이선스 주의점을 안내할 예정입니다.

## 이 장에서 다룰 내용

- 색보정의 기본 — 필터 적용과 밝기·대비·채도 같은 기본 색 조정
- 음악 추가하기 — 영상에 배경음악을 얹는 흐름
- 음악 라이선스 주의 — CapCut 내장 음원을 유튜브 등 외부 플랫폼에 올릴 때의 사용 범위·저작권(Content ID) 문제를 반드시 확인해야 하는 주제, 그리고 로열티프리·YouTube Studio 오디오 보관함 같은 안전한 대안 방향

> 📝 이 장은 **작성 예정**입니다. 특히 음악 라이선스는 플랫폼별 약관·사용 범위를 확인한 뒤 정확히 정리할 예정이며(Phase 22에서 검증된 안내로 채움), 현재는 "확인이 필요한 주제"로만 표시해 두었습니다. 실제 화면 캡처는 트립 후 저자가 직접 캡처합니다.
