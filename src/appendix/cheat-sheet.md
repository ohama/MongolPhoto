# 현장 치트시트

현장에서 빠르게 훑어볼 수 있도록 이 책의 핵심 세팅값·체크리스트만 한 페이지로 모았습니다. 각 값의 이유는 본편 링크에서 확인하세요.

## 1. 시작 세팅값

| 항목 | 시작값 |
|---|---|
| 모드 | M (수동) |
| 조리개 | 렌즈 최대 개방 |
| ISO | 1600~3200 |
| 셔터속도 | 15~20초 |
| 화이트밸런스 | 3800~4200K |
| 파일 형식 | RAW |

최대 셔터속도 공식: `500 ÷ (초점거리 × 크롭계수)` — 자세한 계산은 [500 / NPF 룰](../3-astro/2-fundamentals/500-npf-rule.md)에서, 내 카메라의 크롭계수는 [카메라 · 렌즈 추천 목록](camera-lens-picks.md)에서 확인하세요.

## 2. 별 초점 (요약 스텝)

1. MF(수동초점) 전환
2. 라이브뷰 확대(가장 밝은 별에)
3. 포커스 링 미세조정
4. 필요시 테이프로 고정
5. 테스트 샷 100% 확대 확인

자세한 설명은 [별에 초점 맞추기](../3-astro/2-fundamentals/focusing.md).

## 5. 빠른 문제해결

| 증상 | 대응 |
|---|---|
| 별이 궤적으로 번졌다 | 셔터속도를 500/NPF 상한보다 짧게 재시도 |
| 초점이 흐리다 | 라이브뷰 확대·미세조정을 더 천천히, 테이프 고정까지 반복 |
| 노출이 어긋났다 | 본 촬영 전 히스토그램 확인 단계를 빠뜨리지 않기 |
| 뿌옇게 나온다 | 이슬·김서림 여부 확인, 렌즈 앞 주기적으로 닦기 |

원인·다음 개선까지 자세한 내용은 [야외 연습 결과 점검](../3-astro/3-practice/field-review.md).
