# 촬영 기초

트래킹 장비 없이 은하수를 찍으려면 카메라 세팅과 현장 요령이 전부입니다. 이 파트는 아래 순서대로 읽고 그대로 따라 하면, 실전(서해안 연습·몽골) 전에 "무엇을 어떻게 눌러야 하는지"를 손에 익힐 수 있도록 구성했습니다.

1. **[노출의 기본: M모드 3요소](exposure-basics.md)** — 왜 M(수동)모드여야 하는지, ISO·조리개·셔터속도를 각각 어떻게 정하는지. 은하수 시작 세팅값도 여기서 확인합니다.
2. **[500 / NPF 룰](500-npf-rule.md)** — 셔터속도를 얼마까지 늘려도 별이 점으로 남는지, 크롭센서 보정 공식으로 직접 계산합니다.
3. **[별에 초점 맞추기](focusing.md)** — 초보자가 가장 많이 망치는 부분. 라이브뷰로 별에 수동 초점을 맞추는 단계별 방법입니다.
4. **[RAW와 화이트밸런스](raw-and-wb.md)** — 현장에서 신경 쓸 최소한의 설정과, 후보정에서 마무리할 부분의 구분.
5. **[은하수 찾기와 타이밍](finding-the-milkyway.md)** — 은하수 코어가 언제·어느 방향에 보이는지, 달의 영향까지 확인하는 법.
6. **[현장 촬영 워크플로](field-workflow.md)** — 도착부터 철수까지, 실제로 몸이 움직이는 순서.

> **초보자 안내:** 처음이라면 순서를 건너뛰지 말고 1→6 순으로 읽으세요. 특히 2(셔터속도 계산)와 3(초점)은 실전 하늘 아래서 처음 시도하면 늦습니다 — [가기 전 집 연습](../03-practice/home-camera-controls.md)에서 반드시 먼저 연습해야 하는 항목입니다.
