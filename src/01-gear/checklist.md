# 출발 전 체크리스트

촬영 나가기 전에 이 목록을 그대로 훑어보세요. 카테고리별로 묶었으니, 출발 전날 밤이나 당일 짐 쌀 때 인쇄하거나 복사해서 하나씩 체크하면 됩니다.

## 카메라 & 렌즈

- [ ] 카메라 바디
- [ ] 광각 렌즈 (은하수 촬영용)
- [ ] 다른 상황용 보조 렌즈(선택)
- [ ] 렌즈/바디 캡, 먼지 제거용 블로어

## 전원 & 저장

- [ ] 예비 배터리 (최소 1~2개 이상)
- [ ] 배터리 충전기
- [ ] 보조 배터리(파워뱅크) + USB 충전 케이블
- [ ] 메모리 카드 (여유 있게 여분 포함)
- [ ] 메모리 카드 케이스 (분실 방지)

## 삼각대 & 릴리즈

- [ ] 삼각대
- [ ] 인터벌 타이머 / 릴리즈 (바디에 내장 기능이 없다면)
- [ ] 삼각대 고정용 무게추 대용품(바람 심할 때 가방 등)

## 방한 & 안전

- [ ] 방한복 (밤 기온 대비 겹쳐 입을 옷)
- [ ] 장갑 (터치스크린 조작 가능한 것이면 더 편함)
- [ ] 레드라이트 헤드램프
- [ ] 보조 배터리 겸용 보온 주머니(예비 배터리는 몸 가까이 보관)
- [ ] 렌즈 히터 / 성에 방지대(선택)

## 앱 & 계획

- [ ] PhotoPills 설치 및 촬영 예정지·날짜 미리 확인
- [ ] Stellarium으로 은하수 방향·시간대 사전 확인
- [ ] 날씨(구름) · 달의 위상 확인 (Windy / Clear Outside 등)
- [ ] 촬영 장소까지 이동 경로·시간 확인

## 기타

- [ ] 손전등(백업용, 흰 빛)
- [ ] 물 · 간식
- [ ] 촬영 장소가 사유지/보호구역이 아닌지 확인
- [ ] 함께 가는 일행과 촬영 시작·종료 시간 공유

> 🔰 **초보자는 이렇게.** 출발 전날 밤 이 목록을 인쇄해 위에서부터 하나씩 체크하며 가방에 넣으세요. 빠뜨리기 쉬운 예비 배터리·메모리 카드·레드라이트 헤드램프 세 가지는 특히 두 번 확인하세요.
