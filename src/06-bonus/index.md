# 보너스 기법

기본 은하수 촬영에 익숙해졌다면 시도해 볼 확장 기법입니다. 여러 장을 이어 붙여 더 넓게 담고(파노라마), 별의 궤적을 그리고(스타트레일), 밤하늘의 움직임을 영상으로 압축하는(타임랩스) 방법을 다룹니다.

- [은하수 파노라마 촬영·병합](panorama.md) — 여러 컷을 이어 붙여 넓은 은하수 아치를 담기.
- [스타트레일(별궤적) 촬영·합성](star-trails.md) — 장시간의 별 움직임을 궤적으로 그리기.
- [밤하늘 타임랩스](timelapse.md) — 밤하늘의 흐름을 짧은 영상으로 압축하기.
