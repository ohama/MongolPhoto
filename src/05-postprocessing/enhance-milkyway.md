# 은하수 강조 보정

[RAW 현상 기본](raw-develop.md)으로 기본 톤을 잡은 뒤 진행하는 2차 패스입니다. 강조는 항상 "적당히"가 원칙입니다 — 과하게 올리면 오히려 사진이 부자연스럽게 티가 납니다.

## 강조 기법

아래 기법은 모두 특정 영역을 따로 오려내지 않는, 사진 전체에 적용하는 단일 슬라이더 수준의 조정입니다. 픽셀 단위로 영역을 정교하게 나눠 편집하는 고급 기법은 이 책의 범위 밖입니다.

- **로컬 대비 / 디테일** — 코어의 구조가 드러나도록 로컬 대비(명료도/Texture 계열 슬라이더)를 소폭 올립니다. 전역 대비를 과하게 올리면 배경 별들이 뭉개진다는 점에 주의하세요.
- **선택적 채도** — 은하수의 청보라·황갈 톤을 살리는 방향으로 채도/생동감을 소폭 올립니다. 하늘 전체 채도를 일괄로 올리지 마세요.
- **밝기 / 커브** — 코어만 살짝 밝히되, 배경 하늘 전체가 회색으로 들뜨지 않도록 유지합니다.

## 과보정 함정

강조를 지나치게 하면 아래 세 가지 문제가 흔히 나타납니다.

- **블랙 뭉갬(크러시된 블랙)** — 블랙 레벨을 너무 내리면 어두운 별이나 배경의 미세한 디테일이 통째로 사라집니다.
- **과채도** — 채도를 과하게 올리면 색 노이즈가 도드라지고 부자연스러운 색 얼룩이 생깁니다.
- **헤일로 / 그라디언트** — 명료도·대비를 지나치게 올리면 밝은 영역 경계에 헤일로(빛번짐 테)나 부자연스러운 밝기 그라디언트가 생깁니다.
