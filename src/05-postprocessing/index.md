# 천체사진 보정 (은하수 후보정)

카메라에서 막 나온 은하수 사진은 대개 어둡고, 노이즈가 많고, 밋밋합니다. 이 단계는 **RAW 현상 → 여러 장 스태킹(노이즈 감소) → 은하수 강조 → 전/후 비교**의 순서로, 그 흐릿한 원본을 은하수 사진으로 완성합니다.

주간 여행 사진의 보정(Lightroom Classic)은 대상과 목적이 전혀 다르므로, 1부 · 여행 사진의 [여행 사진 보정](../13-editing/index.md)에서 따로 다룹니다.

- [RAW 현상 기본](raw-develop.md) — 노출·화이트밸런스·기본 톤을 잡는 첫 단계.
- [스태킹으로 노이즈 줄이기](stacking.md) — 여러 장을 겹쳐 노이즈를 줄이고 신호를 살리기.
- [은하수 강조 보정](enhance-milkyway.md) — 미약한 은하수 신호를 자연스럽게 끌어올리기.
- [전/후 비교로 배우는 후보정](before-after.md) — 실제 전/후로 각 단계의 효과 확인.
