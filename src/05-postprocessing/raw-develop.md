# RAW 현상 기본

후보정의 첫 단계는 찍은 RAW 파일을 현상하는 것입니다. 단, 같은 구도로 여러 장을 찍었다면 이 워크플로를 바로 적용하지 말고 [스태킹](stacking.md)을 먼저 해 노이즈를 줄인 뒤, 그 결과물(1장으로 합쳐진 파일)에 아래 순서를 적용하세요. RAW로 찍어야 하는 이유는 [RAW와 화이트밸런스](../02-fundamentals/raw-and-wb.md)에서, 야간 화이트밸런스 시작값(3800~4200K)은 [밤 노출](../02-fundamentals/night-exposure.md)에서 이미 다뤘으니 여기서는 다시 설명하지 않습니다.

**소프트웨어:** darktable(무료, 기본으로 추천)이나 Lightroom(이미 구독 중이면 그대로 사용)이면 됩니다. 아래 순서는 특정 메뉴 경로가 아니라 화이트밸런스·노출·대비·채도처럼 어떤 RAW 편집기에도 있는 슬라이더 개념으로 서술합니다.

## 현상 순서

1. **화이트밸런스** — 색온도 슬라이더를 [밤 노출](../02-fundamentals/night-exposure.md)에서 다룬 3800~4200K 부근에서 시작합니다. 은하수의 청보라 톤이 살아나는 방향으로 미세 조정하세요.
2. **노출 / 하이라이트·섀도** — 전체 밝기(노출)를 먼저 잡고, 하이라이트를 낮춰 밝은 부분(밝은 별, 인공광이 닿은 전경)의 디테일을 보존한 뒤, 섀도를 올려 전경의 어두운 디테일을 회복합니다.
3. **대비 / 디헤이즈(Dehaze, 대기 안개를 걷어 대비를 살리는 슬라이더)** — 은하수와 배경 하늘의 톤 분리를 살리되 과하게 올리지 않습니다. 은하수를 더 뚜렷하게 강조하는 작업은 [은하수 강조 보정](enhance-milkyway.md)에서 별도로 다룹니다.
4. **색감 / 채도** — 은하수의 청보라 톤 방향으로 채도·생동감(Vibrance)을 미세 조정합니다. 배경 하늘 전체의 채도를 과하게 올리지 않도록 주의하세요.

---

기본 톤을 잡았다면 [은하수 강조 보정](enhance-milkyway.md)에서 강조 2차 패스를 이어가고, 결과는 [연습 로그 템플릿](../03-practice/template.md)에 기록해 보세요.

> 🔰 **초보자는 이렇게.** 여러 장을 찍었다면 먼저 스태킹으로 1장으로 합친 뒤 시작하세요. 그다음 화이트밸런스(3800~4200K) → 노출·하이라이트·섀도 → 대비 → 채도 순서로 딱 이 네 슬라이더만 만집니다. 각 슬라이더는 과하지 않게 조금씩만 움직이세요.
