# MongolPhoto

몽골 고비에서 드론 사진과 은하수 사진 찍기 — 장비부터 촬영, 후보정까지 초보자가 따라 할 수 있는 한국어 mdBook 튜토리얼.

## 📖 책 보기

**[몽골 사진 가이드](https://ohama.github.io/MongolPhoto/)** (GitHub Pages)

## 목차

- **소개** — 드론·은하수 사진의 매력, 이 책의 두 축, 최소 장비, 후보정 과정, 예시 사진
- **1부 · 드론 사진** — 개요·규제 요약·DJI Mini 5 Pro 조작/설정·항공 구도·비행/환경·후보정 + 명소별 항공 가이드
- **드론 참고 자료** — 몽골 드론 규제·허가 · DJI 사양·튜토리얼 · 예시 갤러리
- **2부 · 준비하기** — 장비 가이드 + 기초 촬영법(노출·크롭 셔터룰·초점·RAW·타이밍·현장 워크플로) + 연습 로그 템플릿
- **3부 · 가기 전 연습** — 집 연습 · 야외 연습
- **4부 · 몽골 여행지별 촬영 가이드** — 고비 촬영 일반 원리 + 명소별(차강소브라가·바양작·홍고린엘스·욜링암·바가가즈링 촐로)
- **5부 · 후보정** — RAW 현상 → 스태킹 → 은하수 강조 → 전/후 비교
- **보너스 기법** — 파노라마 · 스타트레일 · 타임랩스
- **부록** — 카메라·렌즈 추천 · 현장 치트시트 · 체크리스트 · 앱·소프트웨어 가이드 · 용어 사전
- **참고 자료** — 촬영 기법 학습자료 · 사용 소프트웨어 · 연습용 무료 RAW·샘플 · 몽골 천체사진 링크 · 몽골 여행 실무 정보

## 개발

mdBook으로 작성되며 `main`에 push하면 GitHub Actions가 빌드해 `docs/`를 커밋, GitHub Pages로 배포합니다.

```sh
mdbook serve src   # 로컬 미리보기
mdbook build src   # docs/ 로 빌드
```

- 책 원문: `src/`
- 배포 산출물: `docs/` (CI가 자동 생성)
- 이미지 최적화: `scripts/optimize-images.sh`
- 기획 문서: `.planning/`
