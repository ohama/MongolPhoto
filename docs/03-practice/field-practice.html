실전 연습 — 노출·초점·구도

집에서 손에 익힌 조작을 이제 실제 밤하늘 아래에서 확인할 차례입니다. 현장 촬영 워크플로의 순서를 그대로 따라가면서, 아래 항목을 특히 눈여겨보세요.

근교 어두운 장소 찾기

차로 30분~1시간쯤 벗어난, 도심 불빛에서 최대한 먼 지점을 하나 골라두세요. 그곳의 실제 Bortle 등급을 몽골 고비(Bortle 1~2)와 비교해 보면서 기대치를 미리 잡아두는 것이 좋습니다 — 은하수 찾기와 타이밍의 광해/Bortle 항목을 참고하세요. 장소·시각 확인에 쓰는 PhotoPills·Stellarium·광해지도는 필수 액세서리 · 앱에서 이미 소개했으니 여기서는 다시 설명하지 않습니다.

실전 노출 테스트

실제 밤하늘 아래에서 고ISO(밤 노출에서 잡은 ISO 1600~3200 시작값 참고)로 시험 촬영을 해 보세요. 셔터속도 상한은 500 / NPF 룰 또는 카메라 조작·세팅 연습에서 이미 계산해 둔 본인 값을 그대로 사용합니다(여기서 숫자를 새로 계산하지 않습니다). 집에서는 실내조명이나 먼 불빛으로 다이얼 감각만 익혔지만, 여기서는 실제 별을 촬영해 노출과 히스토그램을 직접 확인한다는 점이 다릅니다.

별 흐름 실측 — 이 연습의 핵심

찍은 프레임을 LCD에서 100% 확대해, 별이 점으로 남았는지, 짧은 궤적으로 번졌는지 직접 눈으로 확인하세요. 번졌다면 셔터속도를 더 짧게 잡고 다시 찍습니다. 집 초점 연습에서는 점광원에 초점만 맞췄지 별이 실제로 흐르는지는 확인할 방법이 없었습니다 — 이 실측이 집 연습과 야외 연습의 가장 큰 차이입니다. 초점 자체가 맞았는지는 별에 초점 맞추기의 방법대로 별도로 확인하세요.

실전 구도

집에서는 창밖이나 베란다로 구도 잡기를 연습했지만, 이제 실제 전경(능선·나무·구조물 등)과 하늘을 세로·가로로 각각 잡아 보세요. 삼각대 수평을 맞추는 법은 초점·구도·인터벌 연습에서 이미 다뤘으니 여기서는 실전 적용만 하면 됩니다.

이 연습 결과는 연습 로그 템플릿에 기록해 보세요.

🔰 초보자는 이렇게. 근교 어두운 곳에서 ISO 1600~3200과 집에서 미리 계산해 둔 셔터속도 상한으로 시험 촬영하세요. 찍은 프레임을 LCD에서 100% 확대해 별이 점으로 남았는지 확인하고, 궤적으로 번졌으면 셔터속도를 더 짧게 잡아 다시 찍으세요.
