# Requirements: MongolPhoto — v1.1 드론 사진

**Defined:** 2026-07-13
**Milestone:** v1.1 드론 사진 (책을 "몽골 사진 가이드"로 확장)
**Core Value (승계):** 초보자가 따라 하며 몽골에서 실제로 성공하도록, 드론 사진(주간·황혼)과 은하수 사진의 전 과정을 담는다.

> v1.0(은하수)은 SHIPPED. 이 마일스톤은 **드론 파트를 은하수 앞에 추가**하고 책을 "몽골 사진 가이드"로 확장한다. 드론 = DJI Mini 5 Pro, 사진 중심(+영상 간략), 주간·황혼 중심(야간 은하수 드론 촬영은 법[주간만]+하드웨어[Mini 한계]로 제외). 규제는 **핵심만 간략** + "출발 전 CAAM/현장 재확인" 정직 표기.

## v1.1 Requirements

### 책 확장 (BOOK)

- [x] **BOOK-01**: 책을 "몽골 사진 가이드"로 확장 — 제목(book.toml)·소개(introduction)·README를 드론+은하수 포괄로 재구성
- [x] **BOOK-02**: SUMMARY 재편 — 드론 파트를 은하수 파트 앞에 배치(기존 은하수 파트 내용은 그대로, 파트 표시 번호만 조정)

### 드론 사진 (DRONE)

- [x] **DRONE-01**: 드론 사진 개요·매력 + 이 파트 범위 명시(사진 중심·주간/황혼, 야간 제외 이유: 법+하드웨어)
- [x] **DRONE-02**: 드론 규제·안전 (간략) — 핵심 규칙(주간·VLOS·고도 상한·공항 이격) + "출발 전 CAAM/현장 재확인", 250g 면제·국립공원 규정은 **미확인** 표기, 상세는 참고 자료로
- [x] **DRONE-03**: DJI Mini 5 Pro로 사진 찍기 — 기본 조작 + 카메라 설정(DNG RAW·노출·ND 필터·히스토그램)
- [x] **DRONE-04**: 항공 구도(탑다운·45°·리딩라인·스케일) + 비행 기초(호버·오빗·리빌)·배터리/RTH 관리
- [x] **DRONE-05**: 고비 드론 촬영 환경 주의(바람·모래·저온·원격지 충전) — 기존 야외 위험 챕터 재활용/링크
- [x] **DRONE-06**: 드론 사진 후보정 — 기존 RAW 현상 챕터 링크 + 드론 델타(렌즈 왜곡·먼지 얼룩·낮은 노이즈)

### 명소별 드론 촬영 (DSITE)

- [ ] **DSITE-01**: 명소 5곳(차강소브라가·바양작·홍고린엘스·욜링암·바가가즈링 촐로) 항공 구도·피사체 가이드
- [ ] **DSITE-02**: 명소별 드론 주의사항 — 국립공원 규정 미확인 → 현장 확인(욜링암·홍고린엘스), 지형별 위험

### 드론 참고 자료 (DREF)

- [ ] **DREF-01**: 몽골 드론 규제·허가 참고 링크 (CAAM 등, 검증/미검증 구분)
- [ ] **DREF-02**: DJI Mini 5 Pro·장비·촬영 기법 튜토리얼 링크
- [ ] **DREF-03**: 드론 항공사진 예시 갤러리 (가능하면 몽골/고비, 라이선스 표기)

## Future Requirements (이후 마일스톤)

## Out of Scope

| Feature | Reason |
|---------|--------|
| 드론 야간·은하수 촬영 | 몽골 법(주간만) + Mini급 하드웨어 한계(짧은 노출·진동) 이중 제약 |
| FPV/레이싱 드론 | 이 책은 여행 항공 사진용 |
| 시네마틱 영상 제작 심화 | 사진 중심 — 영상은 간략 언급만 |
| 드론 구입/구매 가이드 | (v1.0 원칙 승계) 추천만, 구매 강제 아님 |
| 몽골 드론 법률 자문·확정 해석 | 규제는 핵심만 간략 안내 + 공식 재확인 위임 (미확인은 미확인으로 표기) |
