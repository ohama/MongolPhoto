# Milestone v1.3: 여행 사진 보정 (Lightroom Classic)

**Status:** ✅ SHIPPED 2026-07-15
**Phases:** 15-19
**Total Plans:** 9

## Overview

여행 사진 파트(1부) 바로 다음에 **"여행 사진 보정"**(신규 `src/13-editing/`, 9개 챕터)을 넣어, 찍은 여행 사진을 **Lightroom Classic 기준**으로 실제로 더 낫게 만드는 전 과정을 초보자가 따라 하게 담았다. 흐름은 개념·카탈로그·도구 선택 → 표준 편집 순서(17단계 척추)와 상황별 레시피(+CC0 before/after) → 마스킹·병합 → 촬영 연동·크레딧이며, 기존 darktable 은하수 스태킹 파트와 명확히 구분한다(천체 스태킹은 anti-feature, 기존 "6부 · 후보정"은 "은하수 후보정"으로 개칭). 정직성 원칙 승계 — 검증된 CC0/무료 라이선스만 재호스팅, Adobe UI는 "저자 직접 캡처 예정" 플레이스홀더, 원화 요금·렌즈 프로파일 등 미확인은 "미확인/공식 페이지 확인" 표기.

## Milestone Summary

**Key Decisions:**
- 신규 파트를 여행 사진 다음(드론 앞)에 배치, 기존 폴더 renumbering 없이 SUMMARY만 재편(src/13-editing/ 신설). 기존 "6부 · 후보정"→"은하수 후보정" 개칭으로 명칭 충돌 회피.
- 기준 도구 = Lightroom Classic(구독 전용, Photography 플랜 포함·단독 LR CC 플랜엔 미포함). 무료 대안 darktable(R7 CR3 지원, 기존 은하수 파트와 일관)·영구판 DxO/ON1/Capture One 병기.
- 17단계 편집 순서는 "널리 통용되는 관례"로 정직 표기(Adobe 법칙 아님). 마스킹은 Phase 18 단독 소유(중복 방지).
- before/after는 검증된 CC0(Bernard Gagnon) 사진에서 ImageMagick로 생성, 정직 캡션 4요소(CC0 시연·인위적 플랫 not RAW·트립 후 교체·출처) 필수. 실촬영본 교체는 트립(8/13) 후 과제.
- 정직성 승계 — 원화 요금·삼양12mm/RF50mm 렌즈 프로파일 미확인 표기, Adobe UI 스크린샷 재호스팅 금지, ETTR "R7 대낮 효과 제한적·규칙 아닌 선택".
