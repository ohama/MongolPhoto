# MongolPhoto

## What This Is

트래킹 장비 없이 몽골(2026-08-13 ~ 08-17) 여행에서 은하수·천체 사진을 찍기 위한 한국어 mdBook 튜토리얼. 장비 추천부터 촬영법, 후보정, 그리고 실제 촬영한 사진까지 담는다. 집에서의 연습(1·3·4차)과 7/17 밤 서해안 실전 연습(2차)을 거쳐 몽골 현지 날짜별 촬영으로 이어지는 여정을 하나의 book으로 구성한다. GitHub Pages로 배포해 나와 지인이 함께 보고 따라 할 수 있게 한다.

## Core Value

트래킹 장비 없이도 은하수 사진이 "잘 나오게" 하는 전 과정(장비 선택 → 촬영 → 후보정)을, 초보자가 따라 하며 몽골에서 실제로 성공할 수 있도록 담아낸다.

### Out of Scope

- 적도의/스타트래커 등 트래킹 장비 기반 촬영 — 이번 여행 컨셉이 "장비 없이"이므로 제외
- 딥스카이(장초점 망원경) 촬영 — 휴대성·트래킹 없는 여행 촬영과 맞지 않음
- 인쇄용 실물 책 제작 — 우선 웹(mdBook/Pages)로 시작, 필요 시 이후 검토
- 장비 구입/구매 가이드 (구매 링크·리드타임·구매 결정 프로세스) — 추천만 제공, 구매 강제 아님

## Context

- 여행 일정: 몽골 2026-08-13 ~ 08-17 (밤 촬영 4~5회 기회)
- 몽골 코스 (1안, 바양작 제외):
  - Day 1 (8/13): 차강소브라가(Tsagaan Suvarga) 캠프 숙박
  - Day 2 (8/14): 오전 차강소브라가 → 홍고린엘스(Khongoryn Els, 고비 모래언덕)
  - Day 3 (8/15): 욜링암(Yolyn Am)
  - Day 4 (8/16): 바가가즈링 촐로(Baga Gazriin Chuluu)
  - Day 5 (8/17): 공항 (출국)
  - 모두 고비 지역 다크스카이 명소 — 은하수 촬영에 유리.
- 사용자 보유 장비: 크롭 미러리스 바디 (기종 미확인). 장비는 **추천만** 제공하며 구매를 강제하지 않음 — 보유 장비로도 촬영 가능하게 안내.
- 촬영 컨셉: 트래킹 장비 없이, 가볍고 작은 구성 위주 (도보·여행 이동 고려).
- 대상 독자: 나 + 지인(초보자 포함) 공유.
- 언어: 한국어.
- 배포: GitHub Pages (git origin 이미 https://github.com/ohama/MongolPhoto 로 연결됨).
- **book 서술 방식: 일반 가이드** (2026-07-09 전환) — 저자 개인 연습 일지·여행 결과가 아니라, 독자 누구나 따라 할 수 있는 "집 연습 → 야외 연습 → 후보정 → 몽골 여행지별 촬영" 지침으로 구성. 몽골 코스 1안은 여행지별 가이드의 예시 소재로 활용.

## Constraints

- **Tech stack**: mdBook (Markdown) + GitHub Pages — 폴더에 mdBook 도구/스킬 존재, 웹 공유에 적합
- **Timeline**: 서해안 실전 7/17, 몽골 8/13~8/17 — 이 날짜 전에 관련 준비/연습 콘텐츠가 유효해야 함
- **Hardware**: 크롭 미러리스(APS-C) 기준 — 촬영법·화각 계산이 크롭 센서 기준
- **Portability**: 트래킹 장비 없음, 가볍고 작은 구성 — 장비 추천의 최우선 기준
- **Audience**: 초보자 친화 — 설명은 입문자가 따라 할 수 있는 수준
