## Current Milestone: v1.4 드론 영상 (촬영 + CapCut 편집)

**Goal:** 2부 · 드론 사진에 "드론 영상 촬영" 파트를 신설하고, 영상 편집을 **CapCut 기준**으로 상세한 예제 편집과 함께 담아, 초보자가 드론 영상을 찍고 CapCut으로 완성할 수 있게 한다.

**Target features:**
- 드론 영상 촬영 기초 — DJI Mini 5 Pro 영상 설정(해상도·프레임레이트·D-Log M/HDR·ND 필터·180도 셔터 규칙), 안정적 비행·대표 움직임 샷(리빌·오빗·플라이오버·탑다운·드로니), 고비 환경·규제(주간·기존 규정 승계)
- CapCut 편집 워크플로 — 가져오기·컷 편집·순서·속도(슬로모/타임랩스)·안정화·색보정(D-Log→LUT/Log 변환)·음악 비트 싱크·자막/전환·내보내기 설정
- 상세 예제 편집 — 대표 고비 드론 클립 시나리오로 스텝바이스텝(스토리보드→최종), CapCut UI는 "저자 직접 캡처 예정" 플레이스홀더
- 정직성 — DJI/CapCut 사양·요금 검증(미확인 표기), 음악·소재 라이선스 안내, 실제 촬영본은 트립(8/13) 후 교체

## What This Is

몽골 고비 사막에서 **여행 사진(Canon R7)·드론 사진(DJI Mini 5 Pro)·은하수 사진** 세 가지를 찍기 위한 **한국어 mdBook 사진 가이드**. 여행 사진(주간·황혼 지상 촬영) → 드론 항공 촬영 → 트래킹 없는 은하수 촬영 순으로, 초보자가 따라 하면 실전에서 성공하도록 구성한 **일반 가이드**. GitHub Pages로 배포해 누구나 링크로 볼 수 있다.

## Core Value

트래킹 장비 없이도 은하수 사진이 "잘 나오게" 하는 전 과정(장비 선택 → 촬영 → 후보정)을, 초보자가 따라 하며 몽골에서 실제로 성공할 수 있도록 담아낸다.
