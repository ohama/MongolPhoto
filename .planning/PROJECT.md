## Current Milestone: v1.2 몽골 여행 사진

**Goal:** 드론·은하수 파트 앞에 "몽골 여행 사진" 파트를 새 최상단 1부로 추가한다. **Canon R7 + 삼양 AF 12mm F2.0 + RF 50mm F1.8 STM**(보유) 기준으로, 여행 만능 줌 **RF-S 18-150mm** 추천을 곁들여 카메라 설정·여행 사진 잘 찍는 법·코스 명소별 여행 사진 가이드·참고 자료를 담는다.

**Target features:**
- 여행 사진 개요·매력 + 책 구조 재편(여행 사진을 최상단 1부로, 드론·은하수를 뒤로)
- Canon R7 + 삼양 AF 12mm F2.0(초광각)·RF 50mm F1.8 STM(준망원) 카메라 설정 + RF-S 18-150mm 추천 (주간·황혼 여행 사진)
- 여행 사진 잘 찍는 법 (구도·빛·타이밍·풍경·현장/사람)
- 코스 명소별 여행 사진 가이드 (차강소브라가·바양작·홍고린엘스·욜링암·바가가즈링 촐로)
- 여행 사진 참고 자료 (라이선스 가능하면 예시 임베드, 불가하면 링크로 정직 표기)

**범위 결정:** 지상 카메라·주간/황혼 여행 사진 (야간 은하수는 은하수 파트, 항공은 드론 파트) · 보유 킷 = 12mm(≈19mm 초광각) + RF 50mm(≈80mm 준망원), 중간 영역은 여행 만능 줌 RF-S 18-150mm(≈29–240mm) 추천으로 보완 · 참고 사진은 라이선스 확인 시만 임베드, 불가 시 링크.

## What This Is

몽골 고비 사막에서 **여행 사진(Canon R7)·드론 사진(DJI Mini 5 Pro)·은하수 사진** 세 가지를 찍기 위한 **한국어 mdBook 사진 가이드**. 여행 사진(주간·황혼 지상 촬영) → 드론 항공 촬영 → 트래킹 없는 은하수 촬영 순으로, 초보자가 따라 하면 실전에서 성공하도록 구성한 **일반 가이드**. GitHub Pages로 배포해 누구나 링크로 볼 수 있다.

## Core Value

트래킹 장비 없이도 은하수 사진이 "잘 나오게" 하는 전 과정(장비 선택 → 촬영 → 후보정)을, 초보자가 따라 하며 몽골에서 실제로 성공할 수 있도록 담아낸다.

## Constraints

- **Tech stack**: mdBook (Markdown) + GitHub Pages — 폴더에 mdBook 도구/스킬 존재, 웹 공유에 적합
- **Timeline**: 서해안 실전 7/17, 몽골 8/13~8/17 — 이 날짜 전에 관련 준비/연습 콘텐츠가 유효해야 함
