**stacking.md (EDIT-02 — 스태킹 노이즈 감소):**
- 제목: `# 스태킹으로 노이즈 줄이기`
- 오프닝: 고ISO 밤하늘 노이즈를 줄이는 가장 좋은 무료 방법 = 같은 구도 여러 장을 정렬·평균. 노이즈가 대략 √N로 줄어든다는 한 줄. **유료 디노이저보다 이 무료 스태킹을 1순위로.**
- **전제(이 phase의 델타):** 같은 구도로 삼각대 고정, **10~20장, 각 15~20초** 촬영이 전제. "왜 여러 장을 찍는가"·인터벌 타이머 조작은 재설명하지 말고 [현장 촬영 워크플로](../02-fundamentals/field-workflow.md)의 7번(본 촬영)으로 링크. 프레임 간 구도/초점 변경 없이, **수동 WB 고정**([RAW와 화이트밸런스](../02-fundamentals/raw-and-wb.md) 참고)이 필수라는 점만 명시.
- **stack-first 순서 명시:** 개별 프레임을 현상하지 말고, RAW(또는 일관된 기본 변환) 프레임을 그대로 스태킹 도구에 넣어 1장으로 합친 뒤, 그 결과물을 [RAW 현상 기본](raw-develop.md) 순서대로 후보정한다.
- **Sequator (Windows, 무료)** 초보 절차: 프레임 불러오기 → Base Image(전경 가장 선명한 1장) 선택 → Sky Region 대략 지정 → Freeze Ground(전경 고정) → Start → TIFF/PNG 저장. (메뉴 라벨은 "대략 이런 이름"의 안내 수준.)
