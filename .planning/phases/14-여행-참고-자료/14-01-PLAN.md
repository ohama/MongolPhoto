절차:
1. 임시 디렉터리(scratchpad)에 원본 다운로드. Commons 원본은 `Special:FilePath` 엔드포인트로 받는다. 예:
   `curl -L -A "Mozilla/5.0" -o /tmp/dl/tsagaan-suvraga.jpg "https://commons.wikimedia.org/wiki/Special:FilePath/Tsagaan_Suvraga_(20268223089).jpg"`
   파일명의 공백은 `_`로, 괄호는 그대로 URL 인코딩 없이 curl에 넘겨도 되며(따옴표로 감쌈), 확장자 대소문자(.JPG)는 원본 그대로 사용.
2. 각 파일이 실제 JPG인지 확인(`file /tmp/dl/*` → JPEG image data). HTML 오류 페이지가 받아졌으면 파일명 철자를 14-RESEARCH.md 링크와 재대조(특히 Tsagaan_Suvraga vs Suvarga, Baga_Gazaryn vs Gazriin — RESEARCH가 정정한 철자를 사용).
3. `bash scripts/optimize-images.sh /tmp/dl src/images/travel-gallery` 실행 → 최대 2000px 리사이즈 + EXIF/GPS 제거 + JPEG q82. (출력은 모두 소문자 `.jpg`로 통일됨. 로컬 파일명이 위 표와 맞는지 확인하고, 원본 basename이 다르면 최적화 후 `mv`로 위 로컬 파일명으로 정리.)

주의: 임베드는 **CC0/CC BY/CC BY-SA만**. Unsplash/ARR/Instagram/500px 이미지는 절대 다운로드하지 않는다(다음 태스크에서 링크만).
