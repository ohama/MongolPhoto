**스캔 규칙(정확성이 load-bearing — v1.2 게이트 재현):**
- 대상: `src/**/*.md` 전부(SUMMARY.md 포함).
- 링크 추출: 마크다운 인라인 링크 `](target)` 형태. 참조 링크·이미지 링크 `![](...)`도 로컬 경로면 포함(이미지도 존재해야 함).
- **제외:** 외부 링크(`http://`, `https://`, `mailto:`), 순수 앵커(`#...`로 시작).
- **앵커 분리:** `path#anchor`는 `#anchor`를 떼고 파일 경로 `path`만 검사.
- **경로 해석:** 각 링크는 **그 링크가 있는 파일의 디렉터리 기준 상대경로**로 해석한다. SUMMARY.md는 src/에 있으므로 `11-travel/index.md` → `src/11-travel/index.md`. 본문 `../11-travel/camera-settings.md`는 해당 파일 dir 기준으로 정규화(realpath/normpath)한다.
- **존재 검사:** 정규화된 절대경로가 디스크에 파일로 존재하면 OK, 없으면 dead.

**구현(파이썬 권장 — regex+경로정규화가 안전).** 아래 스크립트를 scratchpad에 저장해 실행한다(경로는 프로젝트 루트 기준 절대경로 사용):
```python
import os, re, glob
ROOT = "/Users/ohama/projs/MongolPhoto"
SRC = os.path.join(ROOT, "src")
link_re = re.compile(r'\]\(([^)]+)\)')
total = 0
dead = []
for md in glob.glob(os.path.join(SRC, "**", "*.md"), recursive=True):
    base = os.path.dirname(md)
    with open(md, encoding="utf-8") as f:
        text = f.read()
    for m in link_re.finditer(text):
        target = m.group(1).strip()
        # 제외: 외부·앵커·빈 값
        if target.startswith(("http://", "https://", "mailto:", "#")) or not target:
            continue
        # 앵커 분리
        path = target.split("#", 1)[0]
        if not path:
            continue
        total += 1
        resolved = os.path.normpath(os.path.join(base, path))
        if not os.path.isfile(resolved):
            dead.append((os.path.relpath(md, ROOT), target, os.path.relpath(resolved, ROOT)))
print(f"TOTAL_INTERNAL_LINKS={total}")
print(f"DEAD_COUNT={len(dead)}")
for d in dead:
    print("DEAD:", d)
```
- **dead가 0이 아니면:** 이 계획은 콘텐츠를 고치지 않는다. dead 목록(어느 파일의 어느 링크가 어느 대상으로 깨졌는지)을 리포트에 그대로 적고 게이트 FAIL로 보고한다.
